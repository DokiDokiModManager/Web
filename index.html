What is Doki Doki Mod Manager?

Doki Doki Mod Manager is a tool that simplifies the installation of mods for Doki Doki Literature Club. Furthermore, it seamlessly manages multiple installations of the game, meaning you don't need to worry about losing your save files.


Doki Doki Mod Manager has a rich set of features that wil make your modding experience more fun. What's more, mod creators can use the SDK to add special features to their mods, such as achievements.
